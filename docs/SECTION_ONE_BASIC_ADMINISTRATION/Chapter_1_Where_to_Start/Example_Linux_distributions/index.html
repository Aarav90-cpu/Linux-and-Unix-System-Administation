Example Linux Distributions

Here’s a slightly deeper dive into the Linux players we’ll be featuring heavily.

Debian

Pronounced deb-ian (named after the late founder Ian Murdock and his wife Debra). It’s one of the oldest, most respected distros on the planet. Run entirely by volunteers, it has a hardcore ideological commitment to open-source freedom. Debian manages three simultaneous releases:

  • Stable: Bulletproof. For production servers.
  • Unstable: Bleeding-edge packages where things will inevitably break.
  • Testing: The awkward middle child.

Ubuntu

Built on top of Debian but financially backed by a company called Canonical. Ubuntu releases new versions twice a year (April and October) named with ridiculous alliterative animal codes (like Zesty Zapus). Every two years, the April release is declared a Long-Term Support (LTS) edition, which guarantees five years of updates. If you're running Ubuntu in production, you should definitely be using an LTS release.

Red Hat Enterprise Linux (RHEL) & Fedora

Red Hat is the undisputed heavyweight champion of corporate Linux. RHEL targets massive enterprises that want 24/7 consulting and support to guarantee their systems don't die. Yes, the code is open source, but you have to pay for the license to run it in production.

Where does RHEL get its software? From Fedora, its community-driven testbed. If software survives the wild west of Fedora, it eventually graduates to RHEL.

CentOS (and other RHEL clones)

CentOS is basically RHEL, but with the Red Hat logos scrubbed off and the price tag set to exactly $0. It’s binary-compatible, meaning if it runs on Red Hat, it runs on CentOS. Many companies use a hybrid strategy: they pay Red Hat for mission-critical database servers, and run free CentOS on their fleet of disposable web servers.

Note: Whenever we mention Red Hat or CentOS in this book, the rule applies to both. We won't keep repeating both names because that gets annoying fast. The same rules usually apply to other RHEL clones like Oracle Linux or Amazon Linux.
