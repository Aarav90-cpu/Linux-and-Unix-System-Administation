1.2 Suggested Background

We’re going to assume you already know your way around a UNIX/Linux system. If terms like cd, ls, and "file permissions" sound like foreign concepts, you might want to grab a beginner's guide first (see our recommended reading on page 28). We won’t be covering the absolute basics here.

The Command Line

We love a beautifully designed GUI as much as anyone. But the reality is that Linux GUI tools for administration are still pretty basic compared to what the underlying software can actually do. If you want to get real work done, you have to be comfortable living in the terminal.

Text Editing

You need to learn vi (or its enhanced version, vim). It's installed on every UNIX system by default, meaning you'll never be stranded without it. It’s fast, powerful, and arguably the single best productivity upgrade you can give yourself. Type vimtutor into your terminal for an excellent interactive intro.

If you're completely stuck, GNU’s nano is a simple "starter editor" with on-screen prompts. Just use it discreetly—some of the more old-school admins might visibly cringe if they catch you using it.

Scripting and Coding

Sysadmins aren’t traditionally software developers, but the industry is changing fast. The best admins today are polyglot programmers who don’t mind picking up a new language when it solves a problem.

For your scripting toolkit, we strongly recommend learning:

  • Bash: The default shell almost everywhere. It’s a pretty primitive programming language, but it’s the ultimate duct tape for holding administrative tools together.
  • Python: A highly readable language with a massive community and a library for almost any task you can imagine.
  • Ruby: Developers often describe Ruby as "beautiful" and "a joy to work with."

Honestly, Python and Ruby will solve the exact same problems for you in administration. Pick whichever one you prefer.

You should also check out expect. It’s not a full programming language, but rather a tool for automating interactive programs. It’s an easy-to-learn glue technology that can save you from writing overly complex scripts.
