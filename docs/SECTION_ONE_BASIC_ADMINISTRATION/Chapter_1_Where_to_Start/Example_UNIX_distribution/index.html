Example UNIX Distribution

Let's be honest: commercial UNIX (like Solaris, HP-UX, or AIX) is slowly fading into the sunset. However, the open-source descendants of BSD (Berkeley Software Distribution) absolutely refuse to die.

FreeBSD

FreeBSD still commands a massive cult following among elite system architects, security purists, and huge tech companies. If you’re streaming a movie on Netflix or messaging on WhatsApp, you're interacting with FreeBSD servers doing insanely heavy lifting.

Unlike Linux (which is technically just a kernel that needs thousands of third-party packages to do anything useful), FreeBSD is a complete operating system. The kernel and the core userland tools are developed together and released under the permissive BSD license. We’ll be using FreeBSD 11.0 as our primary example anytime we need to highlight how the BSD/UNIX side of the world does things.
