1.4 Example Systems Used in this Book

To keep things grounded in reality, we’ve chosen three massively popular Linux families and one legendary UNIX variant as our primary guinea pigs: Debian GNU/Linux, Ubuntu Linux, Red Hat Enterprise Linux (and its clone, CentOS), and FreeBSD.

Collectively, these systems represent the vast majority of what you'll actually encounter when you walk into a modern data center or spin up instances in the cloud.

A Quick Note on Formatting

Most of the information in this book applies universally. However, when we need to show you a command, file path, or configuration that is completely specific to one of these systems, we won't be using confusing margin logos.

Instead, we’ll just put a clearly labeled bar directly on top of the code block telling you exactly which distribution it applies to. Simple, right?
